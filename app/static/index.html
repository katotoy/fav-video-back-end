This is a REST web service API for my FavVideo web-app (https://gentle-wave-56839.herokuapp.com)

Main Components
Core - Python/Flask
Persistence - PostgresSQL
Object Relational Mapping (ORM) - Pony ORM
Authentication/Authorization - JWT, jwt-extend
Pasword Hashing - passlib

 

Available Endpoints:

Endpoint Method Parameters Description
/user GET JWT required returns the member's profile
  POST email, call_sign, password creates a new member profile
/login POST email, password verifies if the given credentials is authorized to access the system
/videos/<video_id> or /videos DELETE video_id, JWT required deletes a video with the given video_id
  GET video_id retuns the information of the video with the given video_id
  POST JWT required, url, title, description creates a new video record
/user-videos GET JWT required returns the list of videos posted by a particular member
/gallery GET   returns all the list of videos available from the database
/like-videos/<video_id> GET video_id, JWT required returns a boolean value if the logged user liked the viewed video or not
  POST video_id, JWT required toggles the value on the database reflecting if the user liked or not a video
/video-like-count/<video_id> GET video_id returns the number of likes for the given video_id
/user-videos-like-count GET member_id returns a summary of the videos posted by the given member_id with the total number of likes per video

*JWT required means that the current logged user (member_id) will be determined by the security header passed during request
